# Hướng dẫn đóng góp cho NubWeather

Cảm ơn bạn đã quan tâm đóng góp cho NubWeather! Dưới đây là các hướng dẫn để quá trình đóng góp diễn ra suôn sẻ.

## 🐛 Báo lỗi
1. Kiểm tra xem lỗi đã được báo chưa trong [Issues](https://github.com/your-username/NubWeather/issues)
2. Tạo issue mới với tiêu đề rõ ràng
3. Cung cấp log và cách tái hiện lỗi

## 💡 Đề xuất tính năng
1. Tạo issue với label "enhancement"
2. Mô tả chi tiết tính năng và lợi ích
3. Chờ phản hồi từ maintainer

## 🔧 Pull Request
1. Fork repository
2. Tạo branch mới: `git checkout -b feature/ten-tinh-nang`
3. Commit changes: `git commit -m 'Add: tính năng X'`
4. Push lên branch: `git push origin feature/ten-tinh-nang`
5. Tạo Pull Request
